## 3. Математические модели

### 3.1 Общая структура транспортной задачи

**Переменные:** $x_{ij}$ - объем поставок от поставщика $i$ к потребителю $j$

**Целевая функция (минимизация):**
$$\min z = \sum_{i=1}^{m}\sum_{j=1}^{n} c_{ij}x_{ij}$$

**Ограничения по поставщикам:**
$$\sum_{j=1}^{n} x_{ij} = a_i, \quad i = 1,...,m$$

**Ограничения по потребителям:**
$$\sum_{i=1}^{m} x_{ij} = b_j, \quad j = 1,...,n$$

**Неотрицательность:** $x_{ij} \ge 0$

### 3.2 Особенности открытых задач

**Задача "Школьное питание" (избыток запасов):**
- Добавлен **фиктивный потребитель** с потребностью 15 единиц
- Стоимость перевозок к фиктивному потребителю: 0
- Смысл: нераспределенный избыток продукции

**Задача "Запчасти" (дефицит):**
- Добавлен **фиктивный поставщик** с запасом 8 единиц
- Стоимость перевозок от фиктивного поставщика: 0
- Смысл: неудовлетворенный спрос потребителей
