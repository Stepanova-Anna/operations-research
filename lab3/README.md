## 3. Математические модели

### 3.1 Прямая задача (общая структура)

**Переменные:** $x_j$ - масштаб программы $j$ ($0 \le x_j \le 1$)

**Целевая функция (максимизация эффекта):**
$$\max z = \sum_{j=1}^{4} c_j x_j$$

**Ограничения по ресурсам:**
$$\sum_{j=1}^{4} a_{ij} x_j \le b_i, \quad i = 1,2,3$$

**Границы переменных:** $0 \le x_j \le 1$

### 3.2 Двойственная задача

**Переменные:**
- $y_i \ge 0$ - теневая цена ресурса $i$
- $z_j \ge 0$ - теневая цена верхней границы $x_j \le 1$

**Целевая функция (минимизация):**
$$\min w = \sum_{i=1}^{3} b_i y_i + \sum_{j=1}^{4} z_j$$

**Ограничения:**
$$\sum_{i=1}^{3} a_{ij} y_i + z_j \ge c_j, \quad j = 1,...,4$$

**Единицы измерения:**
- $y_i$: ед. эффекта / ед. ресурса $i$
- $z_j$: ед. эффекта / ед. масштаба программы

Во всех задачах наблюдается одинаковый паттерн:
- **3 программы** выполняются на 100%

**Причина:** Программы конкурируют за ограниченный бюджет. Наиболее эффективные программы получают полное финансирование, наименее эффективные - частичное.

## 7. Общий вывод

### 7.1 Основные результаты

1. **Во всех задачах бюджет является единственным дефицитным ресурсом**
   - Теневая цена бюджета: 1.9000 - 2.0909
   - Увеличение бюджета на 1 единицу дает прирост эффекта на 1.90-2.09 единиц

2. **Трудозатраты и операционная ёмкость имеют нулевые теневые цены**
   - Эти ресурсы не ограничивают рост
   - Увеличение этих ресурсов не улучшает оптимальный план

3. **Оптимальная структура плана:**
   - 3 программы выполняются полностью (100%)
   - 1 программа выполняется частично (4.5%-21.7%)

4. **Прогнозы по теневым ценам обычно точны**
   - Для небольших изменений ресурсов (≤ 5 ед.)
   - При смене активного ограничения прогноз становится приближенным

### 7.2 Практические рекомендации

**Для всех рассмотренных систем:**
1. **Приоритет №1:** Увеличение бюджета
   - Наиболее эффективный способ повысить общий эффект
   - Каждая дополнительная единица бюджета дает 1.90-2.09 ед. эффекта

2. **Приоритет №2:** Расширение эффективных программ
   - Программы 2 и 3 показывают высокую эффективность
   - Программа 4 требует пересмотра или полной замены

3. **Оптимизация структуры:**
   - Полностью финансировать наиболее эффективные программы
   - Частично финансировать или исключать наименее эффективные

4. **Учет локальности прогнозов:**
   - Теневые цены работают только в окрестности текущего решения
   - При значительных изменениях ресурсов требуется пересчет
