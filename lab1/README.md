## 1. Введение

В ходе лабораторной работы были решены четыре задачи линейного программирования из разных предметных областей:

1. **Муниципальная пекарня** (civil_01) - распределение муки и печного времени
2. **Городской ремонтный участок** (civil_02) - распределение бригады и техники
3. **Комплектование полевых рационов** (military_01) - распределение продуктов и упаковки
4. **Ремонтно-обслуживающий склад** (military_02) - распределение комплектующих и времени

Все задачи решались геометрическим методом с последующей проверкой через `scipy.optimize.linprog`.

---

## 2. Математическая постановка задач

### Задача 1: Муниципальная пекарня
$$\max z = 7x_1 + 5x_2$$
$$\text{при ограничениях: } 4x_1 + 2x_2 \le 40, \quad x_1 + 2x_2 \le 22, \quad x_1, x_2 \ge 0$$

### Задача 2: Городской ремонтный участок
$$\max z = 8x_1 + 7x_2$$
$$\text{при ограничениях: } 3x_1 + 2x_2 \le 24, \quad 2x_1 + 3x_2 \le 30, \quad x_1, x_2 \ge 0$$

### Задача 3: Комплектование полевых рационов
$$\max z = 10x_1 + 7x_2$$
$$\text{при ограничениях: } 3x_1 + x_2 \le 21, \quad x_1 + 2x_2 \le 14, \quad x_1, x_2 \ge 0$$

### Задача 4: Ремонтно-обслуживающий склад
$$\max z = 11x_1 + 8x_2$$
$$\text{при ограничениях: } 2x_1 + 3x_2 \le 24, \quad 3x_1 + x_2 \le 21, \quad x_1, x_2 \ge 0$$

## 4. Проверка через linprog

Для перехода от максимизации к минимизации использовалось преобразование:
$$\max z = c^T x \rightarrow \min (-z) = -c^T x$$

### Код для всех задач:

```python
# Задача 1: Пекарня
c = np.array([-7, -5])
A_ub = np.array([[4, 2], [1, 2]])

# Задача 2: Ремонтный участок
c = np.array([-8, -7])
A_ub = np.array([[3, 2], [2, 3]])

# Задача 3: Рационы

# Задача 4: Склад

```

Результаты всех проверок совпали с геометрическим решением с точностью до вычислительной погрешности.
