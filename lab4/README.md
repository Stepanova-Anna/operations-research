## 3. Математическая модель задачи о назначениях

### 3.1 Постановка задачи

**Переменные:** $x_{ij}$ - бинарная переменная, равная 1, если исполнитель $i$ назначается на задачу $j$

**Целевая функция (минимизация):**
$$\min z = \sum_{i=1}^{n}\sum_{j=1}^{n} c_{ij}x_{ij}$$

**Ограничения:**
$$\sum_{j=1}^{n} x_{ij} = 1, \quad i = 1,...,n \quad \text{(каждый исполнитель назначается ровно 1 раз)}$$
$$\sum_{i=1}^{n} x_{ij} = 1, \quad j = 1,...,n \quad \text{(каждая задача назначается ровно 1 раз)}$$
$$x_{ij} \in \{0, 1\}$$

### 3.2 Особенности метода Венгерского алгоритма

- Решает задачу о назначениях за полиномиальное время $O(n^3)$
- Гарантирует нахождение глобального оптимума
- Применим для квадратных матриц (количество исполнителей = количество задач)

### 5.1 Выездные бригады (civil_01)

**Ключевые наблюдения:**
